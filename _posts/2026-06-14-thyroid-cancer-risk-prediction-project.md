Our Database Systems project was Thyroid Cancer Risk Prediction using Machine Learning. We worked on this project in Google Colab, where we used code, data, and a Machine Learning model. We also prepared a project report and planned a video presentation to explain our work.
This project was meaningful because it was related to a medical problem. However, it was also challenging. One of our biggest difficulties was finding a real-world-based dataset. Dr. Bilal Ahmad wanted us to work on a dataset that had real value, especially because medical data should be reliable and accurate. Finding a suitable thyroid cancer dataset took time, and this made us understand that the right dataset is one of the most important parts of a Machine Learning project.
Another major challenge was connecting the frontend with the backend through FastAPI. We wanted users to enter information through the frontend and receive a prediction through the Machine Learning model. However, the connection was not working properly. Even when some parts looked correct, the complete system did not always give the expected result.
We also used LLM tools for help with code, as suggested by our instructor. However, the code we received was sometimes incorrect, incomplete, or not suitable for our project structure. We learned that AI tools can help, but students still need to understand, test, and correct the code themselves.
These difficulties taught us patience, debugging, teamwork, and problem-solving. We are preparing a video presentation to explain our Google Colab work, model, challenges, and lessons.
