Another interesting experience in our Database Systems lab was creating a website using WordPress. This activity was different from SQL queries and server work, but it helped me understand how websites can be created and managed. WordPress allowed us to make pages, add content, and organize a website in a simple way.
Working with WordPress was enjoyable because I could see the result of my work directly. Unlike a query that gives an output in a database, a website is visible and can be explored through pages. This gave me some practical exposure to web development.
The activity also helped me understand the connection between websites and databases. Many websites store information such as users, posts, comments, products, and content. A database works in the background, while the website gives users a simple interface to interact with that information.
Creating a WordPress website made me more interested in how frontend and backend systems work together. It also showed me that Computer Engineering skills can be used to create useful digital platforms. This experience became especially relevant when we later tried to connect a frontend with a backend in our Machine Learning project.
