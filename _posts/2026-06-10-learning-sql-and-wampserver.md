The practical side of Database Systems was very useful because we worked with SQL and WAMPServer. SQL helped me understand how to create tables, insert records, select data, update information, and delete records when needed. At first, SQL commands looked difficult because every query has a specific structure.
With regular practice, I started understanding how commands work. For example, creating a table requires deciding which columns are needed, what type of data they will store, and which field should be a key. Writing queries taught me to be careful because a small mistake in a command can affect the result.
WAMPServer gave us a practical environment where we could perform database tasks. It made the course feel more real because we were not only reading about databases; we were actually creating and managing them. I learned that databases support many systems that people use every day, including websites, hospitals, schools, businesses, and online applications.
This practical work increased my confidence in Database Systems. It also prepared me for the fourteen lab manuals, where we had to apply these concepts repeatedly and complete tasks carefully.
