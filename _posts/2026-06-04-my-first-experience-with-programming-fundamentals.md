Programming Fundamentals was one of the most important courses of my first semester because it introduced me to coding. Before studying this course, programming looked difficult and confusing. I knew that computers follow instructions, but I did not understand how those instructions were written. In this course, I learned basic concepts such as variables, data types, conditions, loops, and functions.
At first, every new concept seemed challenging. I had to understand how a program takes input, processes it, and gives output. Slowly, I realized that programming is not only about writing code. It is mainly about logical thinking and solving a problem step by step.
Dr. Bilal Ahmad, our Programming Fundamentals instructor, encouraged us to understand the purpose behind each line of code instead of memorizing it. This approach helped me a lot because it made programming feel more practical. When I understood the logic first, writing code became easier.
Programming Fundamentals gave me confidence that I could learn technical skills with practice. It was also the course that later led me toward project work and Machine Learning. My experience with coding was not always easy, but every small program taught me something new.
